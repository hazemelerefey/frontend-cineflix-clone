> A streaming-platform-inspired movie discovery interface built around dynamic content loading, carousels, detail views, and entertainment-focused UI design.

---

## Overview

This project explores frontend patterns commonly used in media platforms: hero sections, horizontally scrolling content, modal details, search, and watchlist behavior.

The aim is to create a cinematic browsing experience that still feels smooth and structured.

---

## Product Use Case

Users want to:
- browse trending movies and shows quickly
- explore categories through visual carousels
- open details without leaving the main flow
- search content without friction
- save interesting titles for later

---

## UI Highlights

- carousel-based content exploration
- API-driven movie and TV data
- modal details for titles and metadata
- debounced search behavior
- watchlist interaction using local storage
